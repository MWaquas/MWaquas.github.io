Fishing Spots ON

Swift / Xcode / Python / UI–UX Design

← Back to portfolio
Overview

Fishing Spots ON is an iOS application that maps Ontario’s stocked fishing locations using processed geospatial datasets, providing anglers with a clear and reliable way to discover productive fishing sites.

Design Motivation

While public fish stocking data exists, it is scattered across multiple sources and not easily accessible in a user-friendly format. The goal of this project was to consolidate that data into a fast, intuitive mobile experience.

System Design

The application was built using Swift and Xcode, with Python used to clean, manipulate, and map raw stocking datasets to precise geographic coordinates. This enabled accurate in-app location displays and consistent performance across devices.

Development & Testing

Development was carried out independently from concept to launch. Beta testing was conducted with a group of five anglers, where feedback was collected on UI/UX layout, menu structure, and application responsiveness. Iterative changes were made to reduce hang times and improve overall usability.

Outcome

The app was deployed to the App Store for $0.99 CAD and achieved a Top 12 ranking in the App Store’s Paid Travel category within the first week. The project demonstrates full-cycle ownership, from data processing and software development to deployment and user feedback integration.

Next Steps

Future development will focus on updating the application to support Liquid Glass display and expanding functionality to include wayfinding features that guide users through their environment.
