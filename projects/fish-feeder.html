Overview

This project focuses on designing an automated fish feeding system to eliminate inconsistent manual feeding schedules.

Design Motivation

Many aquarium owners rely on memory or manual schedules, which often leads to missed or inconsistent feedings. Also, many owners do not have anyone to feed their fish when they are away from their home for a long period of time.

System Design

The system uses an Arduino-based controller to drive a servo motor that dispenses a fixed amount of food at scheduled intervals. An LCD display is used as the primary communication interface between the user and the system.

To achieve a compact and modular design, the system was implemented using two custom-designed PCBs. The first PCB serves as a keypad interface using tactile pushbuttons, while the second (main) PCB houses the microcontroller and manages all core system functionality.

Current Status

Version 1 successfully validates the core concept. Future work includes further size reduction, and evaluating the feasibility of small-scale deployment through local aquarium retailers.
