Overview

A collection of three games (Connect Four, Reversi, and Pong) were developed in order to strengthen understanding of algorithms and memory management within C and C++.

Each game implements full board logic, win detection, and real-time input handling.

Outcome

These projects demonstrate clean code structure, responsive gameplay, documentation, and efficient logic execution.
